
## Project Overview

**GigFlow – Smart Leads Dashboard** is a full-stack Lead Management application built using the **MERN stack** with **TypeScript**, designed to streamline lead tracking and sales workflow management. The platform provides a secure and scalable dashboard where users can manage leads efficiently through features like authentication, role-based access control, lead CRUD operations, advanced filtering, search, sorting, pagination, and CSV export.

The application supports two user roles — **Admin** and **Sales User** — with different access permissions to ensure secure and controlled operations. It includes a modern responsive UI, real-time state management, protected routes, loading/error handling, and a clean reusable component architecture for maintainability and scalability.

The project is deployed in production with:

* **Frontend** hosted on [Vercel](https://vercel.com?utm_source=chatgpt.com)
* **Backend API** hosted on [Render](https://render.com?utm_source=chatgpt.com)
* **Database** managed using [MongoDB Atlas](https://www.mongodb.com/cloud/atlas?utm_source=chatgpt.com)

This project demonstrates full-stack engineering practices including secure authentication, RESTful API design, database modeling, deployment, and production-ready code architecture.
